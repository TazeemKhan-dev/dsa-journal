## One-Liner Summary
Maximum difference in an array is obtained by subtracting the minimum element from the maximum element.

## Pattern
Array | Min-Max Scan

## Key Trick
The maximum absolute difference between any two elements always occurs between the global maximum and global minimum.

## Mistake / Insight
Mistake: Trying all pairs with nested loops.
Insight: Absolute difference removes ordering — only extremes matter.
