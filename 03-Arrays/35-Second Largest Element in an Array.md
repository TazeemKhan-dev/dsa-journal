## One-Liner Summary
Find the second largest distinct element by maintaining the largest and second largest values in a single pass.

## Pattern
Single Pass | Two Variables | Greedy

## Key Trick
Ignore duplicates of the largest and only update secondLargest when the value lies strictly between.

## Mistake / Insight
Mistake: Sorting the array or not handling duplicates.
Insight: You only need two variables to track the answer optimally.
